如何加载编译k8s源码

本篇文章主要介绍了如何在windows下通过goland阅读k8s源码。

下载及idea加载GO代码

这边大家可以百度一下，搜一下go的下载及破解版goland的下载，网上教程很多，按照顺序按照下载即可。我这里的版本如下

go: 1.15
goland: 2020.1
